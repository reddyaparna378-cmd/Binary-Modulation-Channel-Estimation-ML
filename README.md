# Binary-Modulation-Channel-Estimation-ML
# Analysis of Binary Modulations Using Channel Estimation with Machine Learning

# Project Description

Wireless communication systems often suffer from noise, fading, and interference which degrade signal quality. Traditional channel estimation methods struggle to adapt to dynamic wireless environments.

This project analyzes binary modulation techniques using machine learning models to improve channel estimation performance.

The system evaluates communication performance using metrics such as Bit Error Rate (BER), throughput, and spectral efficiency.

---

# Objectives

- Analyze binary modulation techniques
- Implement BPSK and QPSK modulation
- Apply machine learning models for channel estimation
- Compare performance of MLP, CNN, and Random Forest models
- Evaluate communication performance under different SNR conditions

# Machine Learning Models

The following machine learning models are implemented:

- Multi-Layer Perceptron (MLP)
- Convolutional Neural Network (CNN)
- Random Forest Regressor

---

# Modulation Techniques

The following binary modulation schemes are used:

- Binary Phase Shift Keying (BPSK)
- Quadrature Phase Shift Keying (QPSK)

---

## Performance Metrics

The models are evaluated using the following metrics:

- Mean Squared Error (MSE)
- Mean Absolute Error (MAE)
- R² Accuracy Score
- Bit Error Rate (BER)
- Throughput
- Spectral Efficiency
